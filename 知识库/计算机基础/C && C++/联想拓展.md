> `#ifndef` `#endif`
> `#ifndef` 是 C 语言中的条件编译指令，用于判断指定的标识符是否未定义。若未定义，则执行 `#ifndef` 与 `#endif` 之间的代码；若已定义，则跳过该代码块。

```c
#ifndef 标识符  // 检查标识符是否未定义
#define 标识符  // 若未定义，先定义该标识符（常与#ifndef配合防止重复编译）
// 代码块（如头文件内容：结构体定义、函数声明等）
#endif  // 条件编译结束
```
